Express.js Portfolio

Node.js 환경에서 Express.js 모듈을 활용한 영화정보제공 API Server

사용 방법

1. https://github.com/kma952727/express.js_project.git
2. npm install
3. npm run dev

사용 기술

0. JS ( No Type Script)
1. Node
2. Express
3. Mysql2
4. Passport & JWT ( Authentication )
6. AWS & Jenkins & GitAction ( CI, CD )
